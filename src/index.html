Help

If you are using a screen reader, some of the keyboard shortcuts described below may not be available.

Input

Use the corresponding keys on the keyboard to input a symbol. The 'clear all' button corresponds to the delete key. The 'left arrow' or 'backspace' button corresponds to the backspace key.

History

Use the 'h' key to open the history.

Click on a history item to load it into the calculation display area.
